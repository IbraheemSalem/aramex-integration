# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-15

### Added
- Initial release
- Multi-merchant Aramex account support
- Shipment creation with queue processing
- Shipment tracking (single and bulk)
- Rate calculator
- Billing system with monthly subscriptions and per-shipment fees
- Free shipment quota management
- SMS notifications (Twilio and Nexmo support)
- Webhook handling with logging
- Dashboard UI (Blade templates)
- Artisan commands for status sync and monthly billing
- Complete API with API key authentication
- Event/Listener system
- PDF label saving
- Comprehensive error handling
- Full logging integration
